## החלטות עיקריות

- **שם המוצר:** Corey (הראשון במשפחת agents).
- **קהל יעד:** Solo freelance recruiters בתחום הטכנולוגי בישראל, וסוכנויות בוטיק (2-5 רקרוטרים).
- **משרות נתמכות:** Backend, Frontend, Fullstack, DevOps, Product. סניוריות Junior to Senior.
- **המוצר:** רשימת פרופילים יומית/שבועית באקסל, למייל. אופציית פניות אוטומטיות (Pro).
- **ערוצים:** טופס באתר + מייל בלבד.
- **תמחור:** ₪990 למשרה ראשונה, ₪690 למשרות 2-4, ₪490 מהחמישית. +₪400/משרה לפניות.
- **שבוע ניסיון חינם** על משרה אחת, ללא כרטיס אשראי.
